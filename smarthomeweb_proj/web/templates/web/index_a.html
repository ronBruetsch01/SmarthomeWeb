Ganz ohne Library kommt diese Startseite aus. Effektiv und einfach wird sie als zweidimensionales CSS-Grid System umgesetzt. Und das ganz ohne das Gefühl, Layouts mit Gewalt hinzubiegen. Einleitung

Was ist überhaupt ein Grid System?

Ein Grid System ist ein Raster, indem Inhalte schnell und einfach plaziert werden können. Wir müssen Content sowohl vertikal als auch horizontal anpassen. Da Grid-Layout auch zu nativem CSS gehört, bietet es sich an, sich die Zeit zu nehmen, um sich rein zu denken und damit warm zu werden.

Grid behinhaltet inzwischen viele Möglichkeiten und verschiedene Herangehensweisen zu stylen (Flow-Layout, Box-Modell). Der einfachheit halber werde ich euch hier nur eine zeigen: Grid-Layout. Aber erstmal ein bisschen etwas zu den Bauteilen die wir benutzen werden.

Luftdruck

Hierbei handelt es sich um den Abstand zwischen den einzelnen Grid Items. (Nicht aber um den Abstand vom Grid Container zu den Grid Items, der wird im Padding des Grid Container festgelegt. Man kann auch nur den Abstand von Rows oder Columns ansteuern, indem man den jeweiligen begriff dazwischen setzt. grid-gap: 15px; grid-column-gap: 10px; grid-row-gap: 20px;

Weiter...

Luftfeuchtigkeit

Hiermit kann man angeben wie breit eine Zelle genau sein soll. Mit der Syntax grid-column: 200px; auf ein Grid Item angewendet, kann man bestimmen, wie breit das einzelne Item sein soll. Gut zu wissen ist auch, dass man angeben kann wo im Grid eine Column aufhören oder starten soll, mit zum Beispiel grid-column: 1 / 3;, kann man angeben, dass die Column sich (bei zwei Spalten) über die gesamte Breite erstrecken soll. Dabei wird die Gap als Element mit gezählt.
